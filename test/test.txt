= Explication
Voici comment on inclue un bloc de code : 
\\<Code:
coco
\\>
Plein de retours à la ligne :



\>
>

La commande \~Remain rappelle un morceau de code, la commande \~Display affiche
un morceau de code (dans le premier cas, on affiche ce qui a déjà été affiché,
dans le second, on affiche le code tel qu'il sera sortit)

Remain : 
~Remain test.py

Display : 
~Display test.py

On peut mettre des blocs de --code **dans un** paragraphe-- (<import os>),
spécifier le langage (<<C>#include \<stdio.h\>>), et échapper les caractères
qui sont utilisés par lpm (<\<\<python\>--test\>>). On peut aussi finir un bloc
de code par \\ en l'échappant : <test \\>.

= Inclusion de fichiers
On peut insérer un fichier sans le parser :

~Include test2.txt

Ou en le parsant :

~Get test2.txt
